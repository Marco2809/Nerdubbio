Musica di sottofondo del video-recap.

Metti qui 5 file audio (loop strumentali, royalty-free / CC0, senza voce),
con QUESTI nomi esatti:

  tense.mp3    -> teso/suspense   (thriller, horror, crime, mystery, war)
  epic.mp3     -> epico/cinematic (sci-fi, fantasy, action, adventure)
  warm.mp3     -> emotivo/dolce   (drama, romance, family)
  light.mp3    -> leggero/allegro (comedy, animation, music)
  neutral.mp3  -> neutro cinematic (default)

Specifiche:
- formato mp3, strumentale (niente voce)
- durata ~30-90s (vanno in loop; meglio se il loop e' pulito, ma non obbligatorio)
- volume/loudness moderato e coerente tra i file
- peso contenuto (idealmente < 2-3 MB l'uno: si caricano nella PWA)
